# hapi-openmusic-api
belajar hapijs dari kelas dicoding belajar membuat aplikasi backend untuk pemula

[queue consumer for send email](https://github.com/ajikamaludin/openmusic-queue-consumer)

requirement 
- node >= 12

### start dev

- install 

   	   npm install 
    
- config database connection

   	   cp .env.example .env
   
- create database table

  	  npm run migrate 
  
- run app

	  npm run start
